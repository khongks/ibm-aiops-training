If you do not have a certificate manager then you must install one. The IBM Cloud PakÂ® foundational services Cert Manager is recommended, and can be installed with the following steps.






<Accordion>
<AccordionItem title="ğŸ’¡ Solution">

1. Run the following command to create an instance of the Certificate Manager.

  ```yaml
  cat << EOF | oc apply -f -
  apiVersion: v1
  kind: Namespace
  metadata:
    name: ibm-cert-manager
  ---
  apiVersion: operators.coreos.com/v1
  kind: OperatorGroup
  metadata:
    name: ibm-cert-manager-operator-group
    namespace: ibm-cert-manager
  ---
  apiVersion: operators.coreos.com/v1alpha1
  kind: Subscription
  metadata:
    name: ibm-cert-manager-operator
    namespace: ibm-cert-manager
  spec:
    channel: v4.2
    installPlanApproval: Automatic
    name: ibm-cert-manager-operator
    source: ibm-operator-catalog
    sourceNamespace: openshift-marketplace
  EOF
  ```


<InlineNotification kind="info">

This creates the Certificate Manager.
